# Netflix Clone using React

Author Urvashi

A fast, minimal, and modern Netflix-inspired interface built with React and Vite. This project focuses on clean UI design, smooth layout structure, and real-time movie data fetched from the TMDB API.

Features
• Modern React + Vite setup
• TMDB API integration
• Featured hero banner
• Category-based movie rows
• Clean hover interactions
• Fully responsive layout
• Simple, scalable architecture
